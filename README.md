
# Flying Intelligence 飞行智能社区

![Logo](FlyingIntelligence_LOGO.png)

---

## 简介 | Introduction
Flying Intelligence（飞行智能社区）致力于低空智能感知、机动、操作与协同等方向的研究与交流。我们聚焦于前沿的飞行智能技术，推动学术与产业的深度融合。

Flying Intelligence is a community dedicated to research and communication in low-altitude intelligent perception, maneuvering, operation, and coordination. We focus on cutting-edge technologies in flying intelligence, promoting deep integration of academia and industry.

---

## 主要内容 | Main Contents

- **主页 | Home**：`index.html`，展示团队介绍、研究方向分布、最新成果等。
- **研究方向 | Research Directions**：低空感知、低空机动、低空操作、低空协同。
- **成果展示 | Publications**：`publication.html`、`files/publications/`，收录团队成员的论文与报告。
- **团队成员 | Group Members**：`group.html`、`groups/`，介绍核心成员与合作者。
- **资源分享 | Resources**：`resource.html`，提供相关数据、工具与学习资料。
- **个人页面 | Personal Pages**：`person/`，团队成员个人成果与介绍。
- **图片与多媒体 | Images & Media**：`files/images/`，用于网站展示的图片资源。
- **样式与脚本 | Styles & Scripts**：`css/`、`js/`，网站前端样式与交互脚本。

## 快速启动 | Quick Start

1. 启动本地服务器（需安装 Python） | Start a local server (Python required):
	```sh
	python -m http.server 8000
	```
2. 打开浏览器访问 | Open your browser and visit:
	```
	http://localhost:8000
	```
