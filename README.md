# Flying Intelligence Community

![Logo](files/images/FlyingIntelligence_LOGO.png)

---

## Introduction
Flying Intelligence is a community dedicated to research and communication in low-altitude intelligent perception, maneuvering, operation, and coordination. We focus on cutting-edge technologies in flying intelligence, promoting deep integration of academia and industry.

---

## Project Structure

- **Entry File**: `index.html` (Home Page)
- **Core Data**: `papers-data.js` (Unified management of all publication data, located in root for easy access)
- **Sub-pages**: `pages/` (Includes Publication, Group, Resource, Research, etc.)
- **Static Resources**:
    - `files/images/`: Image resources (including Logo and paper covers)
    - `files/videos/`: Demo videos
    - `css/` & `js/`: Styles and interaction scripts
- **Automation Tools**: `citationmap.py` (Generates global citation map)

---

## Quick Maintenance

To add or modify publication information, please directly edit **`papers-data.js`** in the root directory.
The Home page and Publication page will be automatically synchronized after any modification.
